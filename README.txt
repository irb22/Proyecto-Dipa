REDWOOD CLIENT PORTAL — PROYECTO DIPA

1. Abre index.html para revisar la página en tu computadora.
2. Para publicarla gratis, sube todos estos archivos a un repositorio de GitHub y activa GitHub Pages.
3. Los datos actuales están dentro de data.js.
4. Esta versión excluye correos, teléfonos, contactos y comentarios internos.

Archivos:
- index.html: página principal
- styles.css: diseño
- app.js: filtros, vistas y funcionalidad
- data.js: datos visibles para el cliente
